Project Introduction

This project presents Chapters 1–3 of Alice in Wonderland by Lewis Carroll, formatted for readability and accessibility. You will find clear navigation, high-contrast typography, and responsive images that wrap within the flow of the text for a book-like experience on the web.

All images and text used in this project are in the public domain. Image credits and licensing are listed in the footer of each page. The design uses a restrained three-color palette and two complementary typefaces to evoke a whimsical yet legible reading experience.

Use the navigation links above—or at the bottom of each page—to jump between the introduction and chapters. Line lengths, line height, and semantic structure are set to support comfortable reading on both mobile and desktop devices.
